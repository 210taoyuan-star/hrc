# Google Search Console 驗證後檢查清單

## ✅ 驗證成功後立即執行

### 1. **提交 Sitemap**
- 在 Search Console 左側選單 → 「Sitemap」
- 新增 Sitemap：`sitemap.xml`
- 提交後等待處理（通常 1-24 小時）

### 2. **請求索引主頁面**
- 使用「網址檢查」工具
- 輸入：`https://www.ownbabytw.com`
- 如果顯示「網址不在 Google 中」，點擊「要求建立索引」

### 3. **檢查重要頁面**
使用網址檢查工具測試以下 URL：
- `https://www.ownbabytw.com` （主頁）
- `https://www.ownbabytw.com/robots.txt`
- `https://www.ownbabytw.com/sitemap.xml`

### 4. **監控索引狀態**
- 前往「涵蓋範圍」查看索引進度
- 檢查是否有任何錯誤或警告

## 📊 預期時程

| 項目 | 預期時間 |
|------|----------|
| 驗證完成 | 立即 |
| Sitemap 處理 | 1-24 小時 |
| 首次索引 | 24-72 小時 |
| 完整收錄 | 1-2 週 |
| 搜尋結果出現 | 3-7 天 |

## 🔍 驗證成功指標

### Search Console 中應該看到：
- ✅ 屬性驗證狀態：「已驗證」
- ✅ Sitemap 狀態：「成功」
- ✅ 涵蓋範圍：開始顯示頁面數據

### Google 搜尋測試：
幾天後可以測試以下搜尋：
- `site:www.ownbabytw.com`
- `"代理孕母" site:www.ownbabytw.com`
- `"台灣代理孕母" ownbabytw.com`
