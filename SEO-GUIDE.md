# SEO 和搜尋引擎提交指南

## 1. Google Search Console 設置

### 步驟 1：驗證網站擁有權
1. 前往 [Google Search Console](https://search.google.com/search-console/)
2. 添加屬性：`https://ownbabytw.com`
3. 選擇驗證方法：
   - **HTML 檔案上傳**：下載驗證檔案並上傳到網站根目錄
   - **HTML 標籤**：在 `<head>` 部分添加 meta 標籤
   - **DNS 記錄**：在域名 DNS 設置中添加 TXT 記錄

### 步驟 2：提交 Sitemap
```
https://ownbabytw.com/sitemap.xml
```

## 2. 其他搜尋引擎提交

### Bing Webmaster Tools
1. 前往 [Bing Webmaster Tools](https://www.bing.com/webmasters/)
2. 添加網站：`https://ownbabytw.com`
3. 提交 sitemap

### 百度搜尋引擎（如需要）
1. 前往 [百度站長平台](https://ziyuan.baidu.com/)
2. 添加網站並驗證
