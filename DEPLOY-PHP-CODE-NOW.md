---

## 📋 **部署步驟（5 分鐘）**

### 第 1 步：進入主題編輯器

1. **登入 WordPress 後台**
   ```
   https://63w.c36.myftpupload.com/wp-admin/
   ```

2. **左側菜單**
   - 點擊 **外觀 (Appearance)**
   - 點擊 **主題檔案編輯器 (Theme File Editor)**

3. **右側找 functions.php**
   - 點擊開啟

---

### 第 2 步：清空現有代碼

1. **全選所有** (Cmd+A / Ctrl+A)

2. **刪除** (Delete 鍵)

3. 編輯器現在應該是**完全空白**

---

### 第 3 步：複製新代碼

**在 VS Code 中：**

1. **打開文件** `functions-final-fixed.php`

2. **全選** (Cmd+A)

3. **複製** (Cmd+C)

---

### 第 4 步：貼上代碼

**回到 WordPress 主題編輯器：**

1. **點擊** functions.php 編輯區域

2. **貼上** (Cmd+V / Ctrl+V)

3. **驗證代碼**
   - 開頭：`<?php`
   - 結尾：`});` （**沒有 `?>`**）

---

### 第 5 步：儲存

1. **向下滾動**找到 **「更新檔案」(Update File)** 按鈕

2. **點擊「更新檔案」**

3. **等待回應**
   - 應該看到「File updated successfully」
   - **不應該看到紅色錯誤訊息**
