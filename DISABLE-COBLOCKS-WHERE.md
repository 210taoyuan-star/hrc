# 停用 CoBlocks 插件 - 逐步指南

## 🎯 目標
停用 CoBlocks 插件以解決兼容性錯誤

---

## ✅ 逐步步驟

### 第 1 步：登入 WordPress 後台

**打開瀏覽器，訪問：**
```
https://63w.c36.myftpupload.com/wp-admin/
```

輸入你的 WordPress 帳號密碼並登入

---

### 第 2 步：找到「插件」菜單

**在 WordPress 後台左側菜單中：**

```
👤 用戶名 (左上角)
├── 📊 儀表板 (Dashboard)
├── 📝 文章 (Posts)
├── 📄 頁面 (Pages)
├── 📸 媒體 (Media)
├── 💬 留言 (Comments)
├── 🎨 外觀 (Appearance)
│   ├── 主題 (Themes)
│   ├── 自訂 (Customize)
│   └── 主題檔案編輯器 (Theme File Editor)
├── 🔌 外掛 ⬅️ 【點擊這裡】
├── 工具 (Tools)
└── 設定 (Settings)
```

**點擊左側選單中的「🔌 外掛」或「Plugins」**

---

### 第 3 步：找到 CoBlocks 插件

**在外掛頁面中：**

1. 你會看到一個搜尋欄
2. 在搜尋欄中輸入：`CoBlocks`
3. 按 Enter 搜尋

**或者直接尋找列表中的 CoBlocks：**
- 插件列表會顯示所有已安裝的插件
- 尋找名稱為「**CoBlocks**」的插件

---

### 第 4 步：停用 CoBlocks

**找到 CoBlocks 後，你會看到：**

```
┌─────────────────────────────────────────┐
│ CoBlocks                                 │
├─────────────────────────────────────────┤
│ 描述: A page builder for WordPress...   │
│ 版本: X.X.X                             │
│                                         │
│ 【停用】⬅️ 【點擊這個按鈕】              │
└─────────────────────────────────────────┘
```

**點擊「停用」(Deactivate) 按鈕**

---

### 第 5 步：確認停用成功

**停用後，你會看到：**

```
┌─────────────────────────────────────────┐
│ CoBlocks                                 │
├─────────────────────────────────────────┤
│ 描述: A page builder for WordPress...   │
│                                         │
│ 【啟用】⬅️ 【現在看到啟用按鈕】           │
└─────────────────────────────────────────┘
```

**✅ 表示 CoBlocks 已成功停用**

---

### 第 6 步：重新整理 WordPress 後台

**按下：**
- **Mac**: Cmd + R
- **Windows**: Ctrl + R

**或點擊瀏覽器的重新整理按鈕 🔄**

---

## 🎯 驗證停用成功

### 檢查 1：進入主題編輯器

1. 左側選單 → **外觀** (Appearance)
2. 點擊 **主題檔案編輯器** (Theme File Editor)
3. 右側找 **functions.php**

**預期結果**：
- ✅ 能正常進入
- ✅ 沒有紅色錯誤訊息
- ❌ 不應該看到 CoBlocks 相關的錯誤

### 檢查 2：檢查 Contact Forms 菜單

**回到左側選單**

向下滾動，應該看到：
```
🔌 外掛
📋 Contact Forms ⬅️ 【我們的表單菜單】
```

**如果看到「Contact Forms」菜單 → ✅ 部署成功！**

---

## 📋 故障排除

### 如果找不到 CoBlocks

**可能的情況：**
1. CoBlocks 已經停用了
2. CoBlocks 沒有安裝

**解決方案：**
- 進入主題編輯器查看 functions.php 中是否有我們的代碼
- 檢查是否出現「Contact Forms」菜單

### 如果停用後還有紅色錯誤

1. 重新整理頁面 (Cmd+R / Ctrl+R)
2. 等待 10 秒
3. 再次進入主題編輯器
4. 如果還有錯誤，告訴我具體訊息

---

## 🚀 成功後的下一步

### 1. 驗證表單代碼

- 進入 **外觀** → **主題檔案編輯器**
- 打開 **functions.php**
- 確認代碼以 `<?php` 開頭
- 確認代碼以 `});` 結尾（**沒有 `?>`**）

### 2. 測試表單提交

- 訪問 https://www.ownbabytw.com
- 填寫並提交聯絡表單
- 檢查 WordPress 後台 → **Contact Forms** 菜單
- 應該能看到你的測試提交

### 3. 驗證郵件通知

- 檢查 qiyunsolution@gmail.com
- 應該收到通知郵件
