# ⚡ 快速修復 - 3 步完成

## 問題
複製 Markdown 代碼區塊時，把 ` ``` ` 標記也複製進去了

## 解決方案

### ✅ 方法 1：從純 PHP 文件複製（推薦）

**在 VS Code 中：**

1. 打開文件 `functions-clean-copy.php`
2. **全選** (Cmd+A)
3. **複製** (Cmd+C)
4. 登入 WordPress → 外觀 → 主題編輯器 → functions.php
5. **全選** (Cmd+A)
6. **貼上** (Cmd+V)
7. 點擊「**更新檔案**」

---

### ✅ 方法 2：如果還是有錯誤

檢查 WordPress functions.php 的末尾：

1. 滾動到最底部
2. 如果看到 `?>` → **刪除它**
3. 如果看到多個空白行 → **刪除多餘的行**
4. 文件應該以 PHP 代碼結尾，**沒有 `?>`**

---

### ✅ 方法 3：終極解決方案

如果還是失敗，用 **FTP** 直接編輯：

1. 用 FTP 軟體連接到 GoDaddy (用戶名、密碼、主機)
2. 導航到 `/wp-content/themes/go/`
3. 右鍵 `functions.php` → 下載到電腦
4. 用文本編輯器（VS Code、Notepad++）打開
5. **刪除全部內容**
6. 從 `functions-clean-copy.php` 複製代碼
7. **貼上**
8. **儲存檔案** (重要：使用 **UTF-8 編碼**)
9. 上傳回 `/wp-content/themes/go/` 覆蓋原文件
