# LINE ID 表單欄位設定指南

## 🆕 新增功能
網站聯絡表單已新增 **LINE ID** 欄位，讓潛在客戶可以提供他們的 LINE 聯絡方式，方便後續一對一諮詢。

## 📋 表單欄位更新

### 現在的表單包含：
1. **姓名** (必填)
2. **Email** (必填) 
3. **聯絡電話** (必填)
4. **LINE ID** (選填) ⭐ **新增**
5. **想了解的重點** (必填)

## 🔧 Formspree 後台設定

### 1. 登入 Formspree
- 前往 https://formspree.io/
- 使用 **jctommyliu@gmail.com** 登入

### 2. 查看表單提交
當用戶填寫表單時，您會在 Formspree 後台看到以下欄位：

```
Name: [用戶姓名]
Email: [用戶信箱]
Phone: [用戶電話]
LineId: [用戶 LINE ID] (如果有填寫)
Message: [用戶訊息]
```

### 3. 郵件通知格式
您收到的郵件會包含所有欄位資訊，包括用戶的 LINE ID（如果提供的話）。

## 📱 LINE ID 規格說明

### 使用者體驗：
- **欄位標籤**: "您的 LINE ID (選填)"
- **輸入提示**: "@abc123" 格式範例
- **驗證**: 無特殊驗證，因為是選填欄位

### 常見 LINE ID 格式：
- 官方帳號: `@abc123` (以 @ 開頭)
- 個人帳號: `username123` (可能沒有 @)
- 用戶可以自由輸入任何格式

## 🎯 業務優勢

### 為什麼添加 LINE ID：
1. **台灣用戶習慣**: LINE 是台灣最主要的即時通訊工具
2. **便於跟進**: 可以直接透過 LINE 進行一對一諮詢
3. **提高轉換率**: 提供多種聯絡方式增加客戶回應機會
4. **個人化服務**: LINE 聊天更適合敏感的代孕諮詢討論

### 建議跟進流程：
1. 收到表單後，優先使用電話聯絡
2. 如果電話聯絡不便，可使用 LINE 方式
3. LINE 更適合分享詳細資料、圖片、文件等

## 🚀 技術實作

### 前端更新：
- 新增翻譯常數 `formLineId`
- 在表單中加入 LINE ID 輸入欄位
- 設為選填欄位，不影響表單提交

### 後端處理：
- Formspree 會自動接收 `lineId` 欄位
- 無需額外設定，與其他欄位處理方式相同
